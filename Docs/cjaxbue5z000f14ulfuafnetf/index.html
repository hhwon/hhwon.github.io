如何优雅地摸鱼

如何优雅地摸鱼？写在研究生第一年即将结束前夕。这里总结下这一年学到了什么，以此自勉！晚上与一番大叔切磋了几局炉石，惊心动魄，留下阴影了（233333，心情很舒畅）。生活也是如此，甘于平淡还是颠簸起伏，我想我可能更倾向于后者。总想写点什么，可是每到敲字时，却挤牙膏似的，同时，作为万年挖坑王，又总想给自己挖个填不好的坑（多阅读点文学作品，在信息时代日益发达的今天，静下心来品读点文学作品，感觉炫酷到没朋友）。再说说Minecraft，跟风入坑，随后入正（从大四开始，我的版权观逐步形成了，现在PC/Mobile上几乎没有盗版软件（包括正版方式实现的盗版），对于这个问题，我想是所处的大环境影响的，在步入大学前，尽管喜欢在PC上折腾，但是所处的环境是国内各种不入流的社区，对待破解二字，以前一点感觉都没有，而且用得理所当然，这可能与我一开始的上网姿势有大的关系，同时，支付方式的进步也应该是这一变化的推进剂。我觉得这种观念应该从幼小时期就应该培养着，而不是一面喊着口号，一面却用得冠冕堂皇），到现在和小伙伴一起管理着一个小型服务器（网络上的家）（出于对Minecraft的热爱，以及对这份友谊的延续的憧憬，在这上面，花费了很多，但是也学到了很多，抛开游戏的技术部分不说，这让我对于Linux的操作更加熟练了，同时，对于网路方面的一些知识也有了一点了解）。

渍渍渍，一下子敲了这么多，话痨附体了//////

当然，学业上的收获有点说不出口了，从入学开始说起，一开始就在导师面前立了个flag （问询自己兴趣时，顺口说了对边坡感兴趣（上了假的大学），结果出现了这个大坑，导致现在要挤牙膏似的造一篇小论文（自己做的承诺，含着泪也要实现），我想尽快结束这个，但是心有余而力不足，可能自己没有沉迷学习吧，这个flag伴随了这一整年，想在学习上做点其他的事情，却总是放不下这个，有时候，也验证了长者说的一句话，闷声发大财），敲到这里，我回想下当初读研的初衷，也许就是跟风吧，同时自己的一颗梦想沉迷学习的心，如果（好吧，没有如果），但是我又不想抛弃现在的所学，只能静下心来，迫使自己沉迷学习，同时挤出一点业余时间，培养点广泛的兴趣爱好。作为二次元边缘人（何为边缘人？就是想去接触热爱二次元，但是感觉太老了，难以融进，只好在边缘徘徊），伪技术宅（什么都懂一点点，但是没一样精通），学术渣（想沉迷学术，但是又沉浸不了），。。。。。

写了这么多，感觉跑题了。。。怎么摸鱼呢？

先写这么多，下次话痨附体时继续水=w=
