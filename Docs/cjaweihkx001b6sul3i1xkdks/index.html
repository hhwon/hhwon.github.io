记录这一年的一些事

大概在去年的这个时候，在某种原因下，鼓起勇气开始接触了VPS，以前听到服务器这个词，总以为是一种触摸不到的东西。为了能够接触到一些神奇的东西，第一次入手的则是Bandwagon，这个被国人戏称为搬瓦工的乐队花车，一脸迷惘的我没有得到很好的体验，便弃了。后来在一些网络博文的指导下，找到了DigitalOcean，折腾下能够偶尔获得好的体验，因为没有黑科技的加成。。到了今年，已经对linux基础有点了解了，为了充分让VPS充分利用，投向了价格屠夫Rectified。VPS确实让我感受到了很多神奇有趣的事情，但是又由于拖延症的缘故恐无时间好好学学。
第二件事应该是沉迷Minecraft吧，大约在2014年入坑的Minecraft，一开始知晓是通过某之家的介绍，然后最先接触了手机版，可是却一脸懵逼地不知如何感受游戏的乐趣。后来，在某人的指导下，让我重燃起对于Minecraft的乐趣。真正沉迷应该是在去年毕业摸鱼找到服务器游戏时，一开始的时候，只想默默地潜水，后来被海鸥占了家，这个超级奶爸带着儿子和我开始了服务器之旅。。之后又结识了一些人，玩了服务器原版生存，几乎从零开始体验了熟知了一遍wiki。在暑假之后又用心玩过一个服务器，只是由于某种原因，至今还留着坑呢。最后在服务器中感受聊天的乐趣。。当然，在沉迷游戏中结识到了一群很棒的人，也有一些不棒的人。
还有一件事就是搭建了自己的个人blog等一系列个人website（具体的子域名当然不透露），学到了很多关于网络、linux等的知识。
最后，总结下这一年挖的坑（不包括填好的）1.至今无结果的PFC数值模拟（已弃的状态） 2.还未填好的挡土墙的坑（填坑中，尽量在下星期填得差不多） 3.坑越来越多的minecraft的坑（生生不息，填坑不止） 4.不止如何总结的各种坑。。。
睡了，明天又是美好的一天—-2017/3/19 23：15
