July 9, 2017

Impressions of some news

发牢骚的专用今早读到一则新闻，一个5岁的小男孩因为嘻戏将比他年幼的小女孩独自留在了电梯并升至顶层，由于顶层的护栏问题，小女孩不幸坠亡。我的反应是对小女孩感到惋惜与不幸，其次是对小男孩的未来感到深切的同情与担忧，再则是对该小区的物业维护感到不满。怀着好奇心我特意对这则消息的评论进行了留意，让我感到意外的是，满屏是对小男孩的品行、教育的痛责以及各种恶毒的话语。再看了看消息的标题，小男孩被冠以熊孩子的帽子，这就是我们社会的舆论导向，不对真正应该负责任的物业进行谴责与追责，而将大多数的责骂施加于一年仅5岁的小男孩身上。同时可想而知，如果不对小男孩进行心理及身理的保护，那么小男孩的未来可能将生活在这件事的阴影里，这将是更可悲的事情吧！（不多说），这也让我想到了一部关于这方面的电影，引用豆瓣里的一段影评，“一个横跨十七年的故事，一个中国年轻囚犯的狱中狱外，十几二十岁如花的年华荒废在死灰般的监狱中，这么鲜活的一条生命被压抑在八九十年代脆弱刻薄的社会观念里“。几十年后的今天的社会还是如此的脆弱刻薄吗？这也让我想到了一人说的，我们的社会，物质文明是上来了，但是精神文明任重而道远。

透过这件事，我又想起了前几天看的一则TED演讲，”Addiction is a disease. We should treat it like one”,讲到在美国，对待上瘾的人由原来的逮捕变为现在的以病人对待，这正是社会的包容的进步。想想我们的国家，这方面的观念确实还比较落后。（话痨没附体，这段只有这么长了）

愿世界更加包容！
